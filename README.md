# Adwoa Abban · Quantum Phases Consortium — Digital Business Card

Mobile-first digital business card that opens when someone taps the NFC card.
Uses the Quantum Phases brand colours (teal / red / olive).

## What's inside
`index.html`, `404.html`, `asset-manifest.json`, `static/` — the whole site.
Fully static, hosts anywhere.

## Quick actions the card supports
- Call, Text, Email
- Save contact (.vcf)
- Website, Directions (Google Maps)

## Host on GitHub Pages (free)
1. Create a public GitHub repo, e.g. `quantum-card`.
2. Upload every file from this folder to the `main` branch.
3. Settings → Pages → Deploy from a branch → `main` → `/ (root)`.
4. Your live URL: `https://<username>.github.io/quantum-card/`
5. Write that URL to the NFC card using any NFC writer app.

## Brand colours
- Teal `#0d5f6e` · Red `#c8262e` · Olive `#b4c22e`
- Fonts: Playfair Display (headings) + Inter (body)

Open the site itself for a friendlier `/setup` page with step-by-step
instructions and alternative hosts (Netlify Drop, Cloudflare Pages, Vercel).
